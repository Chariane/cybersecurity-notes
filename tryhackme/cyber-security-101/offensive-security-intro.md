# Offensive Security Introduction

## Objectif
Comprendre comment les hackers éthiques trouvent des vulnérabilités dans les systèmes.

## Lab : FakeBank

Dans cet exercice j'ai :

1. Cherché des routes cachées avec l'outil `dirb`
2. Découvert des routes non protégées
3. Accédé à ces pages
4. Ajouté de l'argent à mon compte

## Outil utilisé

dirb : outil permettant de découvrir des dossiers cachés sur un serveur web.
